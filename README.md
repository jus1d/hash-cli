# hash-cli-tool

Hash CLI Tool is a terminal utility to fast hashing strings or files, checksums. 

## Quick usage:
### For strings:
```bash
hash-cli [ hash-algorithm ] [ raw-input ]
```
### For files:
```bash
hash-cli [ hash-algorithm ] --file [ file-name or path ]
```
